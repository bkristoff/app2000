# APP2000 Applikasjonsutvikling for web
Undervisningsmateriale til emnet APP2000 Applikasjonsutvikling for web, som jeg underviser ved Universitet i Sørøst-Norge (USN), campus Bø.

Dette prosjektet / repo'et er opprettet for å demonstrere / utforske mulighetene i GitHub og inneholder utvalgt eksempelkode fra noen av leksjonene i dette emnet. Resten av materialet ligger i læringsplattformen Canvas (kun tilgjengelig for USN-studenter).
