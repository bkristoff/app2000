2. JavaScript: DOM-en og event-håndtering (https://dbsys.info/2000/leksjon02/index.html)
3. JavaScript: Funksjoner og objekter + JSON (https://dbsys.info/2000/leksjon03/index.html)
4. Versjonskontroll med Git og GitHub (https://dbsys.info/2000/leksjon04/index.html)
5. Smidig applikasjonsutvikling med Scrum (https://dbsys.info/2000/leksjon05/index.html)
6. Bygge UI med React (https://dbsys.info/2000/leksjon06/index.html)
7. Bygge REST API med Express (https://dbsys.info/2000/leksjon07/index.html)
8. Databaseprogrammering: NoSQL, MySQL og ORM (https://dbsys.info/2000/leksjon08/index.html)
9. Single Page Web Applications (SPA) (https://dbsys.info/2000/leksjon09/index.html)
10. Sikkerhet (https://dbsys.info/2000/leksjon10/index.html)
11. Testing og DevOps (https://dbsys.info/2000/leksjon11/index.html)
