# Nettbutikk (fullstack)

Vi utvider nettbutikken med autentisering basert på JavaScript-biblioteket Passport.

Autentisering gjøres med brukernavn og passord, dvs. "local strategy" som det kalles i Passport.

Jeg tok utgangspunkt i følgende tutorial: https://www.passportjs.org/tutorials/password/

Den er imidlertid ikke tilpasset React (bruker såkalt EJS for å bygge brukergrensesnittet), så det ble noen endringer.

Jeg tok utgangspunkt siste versjon av butikk-tjeneren mot MySQL og siste butikk-klient lagd med Vite.

Se README-filene på undermappene for flere detaljer.

Her er et par andre eksempler på bruk av Passport:

- https://github.com/vercel/next.js/tree/canary/examples/with-passport
- https://github.com/passport/todos-express-password
