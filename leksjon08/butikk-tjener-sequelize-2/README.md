# Utvid Express + MySQL + Sequelize

Jeg har ikke laget noen "oppskrift" til dette prosjektet, men det å kode "varianter" av eksisterende kode er ofte en effektiv måte å lære på.

Først må du bygge prosjektet på samme måte som butikk-tjener-sequelize-1.

Prøv deretter å utvide dette prosjektet slik:

- Lag modellklasse for en ny tabell i Hobbyhuset (se mappe \models)
- Utvid med tilhørende databasefunksjoner (se \models\index.js)
- Utvid med nye ruter
- Lag og kjør testfiler
