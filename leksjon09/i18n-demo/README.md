# Flerspråklighet med i18next

Denne mappen demonstrerer hvordan å få til flerspråklighet med biblioteket i18next (https://www.i18next.com/).

Tar utgangspunkt i en minimal React-klient med Vite som byggeverktøy.

Språkfilene ligger på public\locales (norsk og engelsk).

Den interessante koden ligger på i18n.js.

Appen viser en velkomsttekst på engelsk, som byttes ut med norsk tekst med et knappeklikk.

Kjør som vanlig med npm i og deretter npm run dev.
