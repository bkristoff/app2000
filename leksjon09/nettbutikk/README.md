# Nettbutikk (fullstack)

Vi kombinerer butikk-klient-4 (Vite-varianten) fra leksjon 6 med butikk-tjener-3 med database (MySQL-varianten) fra leksjon 8.

Utvidelsen blir å hente varedata fra databasen med Axios-kall og useEffect hook.

Se spesielt på butikk-komponenten og handlekurv-komponenten i klienten og på databasekoden og server.js i tjenerdelen.
